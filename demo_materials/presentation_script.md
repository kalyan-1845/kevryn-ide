## 🏛️ Part 2: Faculty Command Center (Walkthrough)
**Visual**: Point to the Sidebar and Dashboard cards.
**Script**:
> "As you can see, the **Faculty Command Center** gives instructors total control. Let’s walk through the core features:"

*   **1. My Courses & Assignments**: "Instructors can manage full student rosters and distribute assignments directly. No more manual file sharing—everything is automated."
*   **2. Aptitude Modules**: "We’ve integrated a Live Aptitude Test system, allowing the college to conduct placement training and assessments directly within the IDE."
*   **3. Gradebook**: "Every score from every lab is centralized here, giving a clear picture of student performance over the entire semester."
*   **4. Live Monitor (The Heart of Kevryn)**: "This is our most powerful development. It allows the instructor to see every student's code live. If a student switches tabs or pastes code from outside, the system flags them immediately. It ensures 100% academic integrity."
*   **5. Forensic Reports**: "Finally, we can generate instant PDF reports. These aren't just scores—they are detailed activity logs showing exactly how much time a student spent on each problem."

## 💻 Part 3: Student Workspace (Walkthrough)
**Visual**: Switch to the Student IDE view.
**Script**:
> "Now, let's look at the student side. Our goal was to create a professional environment that feels like a top-tier tech company."

*   **The Workspace**: "A full-featured IDE with a high-performance editor and a built-in terminal. Students don't need to install anything—they just log in and start coding."
*   **My Learning**: "Students can see their assignments, course materials, and progress in one dedicated view. It keeps them organized and focused."
